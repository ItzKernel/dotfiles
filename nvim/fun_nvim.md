## Fun nvim commands and stuff
These are fun commands that i wanna keep using without forgetting them lolol

### General

- `C-o`: Switch to normal mode for only one command
- `C-d`: Scroll down faster

### Splits

- `<C-w> <v>`: Split screen vertically
- `<C-w> <s>`: Split screen horizontally
- `<C-w> <_>`: Max out height of selected split
- `<C-w> <|>`: Max out width of selected split
- `<C-w> <=>`: Normalize (equalize) all split sizes
- `<C-w> <R>`: Swap 2 splits


### Tabs

- `:tabnew`: Create a new empty tab
- `:tabe (filename)`: Edit a file 
- `gt`, `gT`: Move between tabs
